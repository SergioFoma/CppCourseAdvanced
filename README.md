# Описание домашних работ 📚

## Arithmetic Operations

**цель  работы**: научиться эффективно извлекать вещественный корень из вещественного числа

* В файле arithmetic_operations.pdf выведена фомрула для расчета
* В фале arithmetic_operations описана сама функция извлечения корня

## String

**цель работы**:  реализовать свой строковый класс (std::string).

Реализованы:

* 9 конструкторов
* 35 методов
* перегружены операции сложения (как внешние функции от двух параметров)
* 1 деструктор

# Установка репозитория 👾

### **Чтобы скачать весь проект, нужно:**

1. В Вашем терминале необходимо указать директорию, куда вы хотите скачать данный проект.

2. Затем в терминале введите:
```
git@github.com:SergioFoma/CppCourseAdvanced.git
```

3. Дождитесь скачивания.

4. После чего будет создана рабочая директория с названием репозитория.

# Запуск программы 🚀

1. Для запуска какой - либо из домашних работ необходимо перейти в папку этого задания.
```
cd String
```

2. После этого для сборки проекта необходимо прописать команду:
```
cmake -S. -B Build
```

, где ```Build``` - папка, где будет создан Makefile и сопуствующие файлы.

3. Для сборки исполняемого файла необходимо прописать команду:
```
cmake --build Build
```

4. Запуск программы:
```
Build/string
```
